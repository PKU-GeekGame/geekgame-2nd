## 【我用108天录了个音】补充说明

（发布时间：2022-11-20 12:04:32）

<ul>
<li>根据<a target="_blank" rel="noopener noreferrer" href="https://cloud.tencent.com/document/product/1093/46740">腾讯云的相关文档</a>，音频应该编码成单声道 8000Hz 采样率，否则效果不佳。</li>

</ul>
